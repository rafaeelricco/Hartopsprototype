Operators can now manage educator assignments without leaving the events list. A new **Educators** column shows assignment status at a glance, and quick-action buttons let you assign or reassign educators inline.

## Quick-Action Buttons

Each event row now includes an action button on the right:

- **Unassigned events** (Upcoming/Live) show a **+** icon to open the assignment dialog
- **Assigned events** (Upcoming/Live) show a **pencil** icon to reassign educators
- **Completed/Cancelled events** have no action buttons since assignments are locked

Clicking the action button opens a dialog without navigating away from the list.

## Assignment Dialog

The dialog shows all active educators with their quality score, location, event count, and specialties. Search by name to quickly find the right person, select one or more educators, and confirm.

## Assignment Status Filter

A new **All Assignments** filter dropdown joins the existing filters, allowing operators to quickly surface:

- **Assigned** &mdash; events with at least one confirmed educator
- **Unassigned** &mdash; events with no educators assigned
- **Pending** &mdash; events where at least one assignment is awaiting response
