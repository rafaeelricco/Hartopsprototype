# Hart SharePoint — LLM Extractions

Text and Markdown extractions of documents from the Hart Agency SharePoint site, synced via OneDrive.

**Source:** `OneDrive-SharedLibraries-HartAgencyNY / Ambar-HART Teams Site - Documents / Ambar-Development`

## Files

| File                     | Description                                                                                                                                                                                                                                                                                                                                                                         |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `all-documents.md`       | Combined text extraction of all documents from the SharePoint site. Organized into 8 sections: HEMs Master Documentation, Technical Outlines, Process & Questionnaire docs, PDFs, Spreadsheet data, Image descriptions, Presentation data, and the Visio tech vision diagram. Contains summaries of spreadsheet and Visio content — see the files below for full-fidelity versions. |
| `tech-vision-diagram.md` | Mermaid diagram conversion of `HART - TECH VISION Diagram and Platfrom.vsdx`. 12 renderable Mermaid diagrams covering platform strategy, architecture layers, AI email processing, database schema, data warehouse design, and AWS infrastructure. `all-documents.md` has the same content as prose text; this file provides structured diagrams.                                   |
| `xlsx-extraction.txt`    | Raw tabular data extracted from the 4 Excel spreadsheets (Wagner 2025 events, Campari questionnaire answers, HEMs Enhancement Questionnaire schema, Platform Capabilities Matrix). `all-documents.md` has column headers and summaries; this file has every row.                                                                                                                    |

## Source Documents Processed

- 12 `.docx` files (technical outlines, process docs, questionnaires)
- 6 `.pdf` files (evaluation reports, survey decks, database schema, event forms, platform outline, AWS notes)
- 4 `.xlsx` files (event schedules, questionnaire data, capabilities matrix)
- 5 `.png` files (screenshots, event report form templates)
- 1 `.pptx` file (SGWS Fall Holiday Survey)
