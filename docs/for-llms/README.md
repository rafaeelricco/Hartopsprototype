# For LLMs

This folder contains information extracted from external sources and converted into LLM-friendly plain text and Markdown formats. These extractions allow LLMs (e.g., Claude, GPT) to answer questions about documents that would otherwise be inaccessible due to format limitations (Visio, Excel, PowerPoint, etc.).

## Contents

| Folder | Source | Description |
|--------|--------|-------------|
| `hart-sharepoint/` | Hart Agency SharePoint site (via OneDrive sync) | Extracted text from 31 documents covering HEMs application documentation, technical outlines, platform architecture (Visio), event data, questionnaires, survey decks, and database schemas. |
